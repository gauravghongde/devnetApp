What's devnet all about??

A Q&A website, inspired from stack-overflow.
The idea of this project is to create an open source alternative to stack-overflow, with a new age tech stack.
The project is completely open-source and the entire Source code is available for free on GitHub. Please feel free to use & do help me to make the product better and robust by Contributing!

Tech Stack Used :
Springboot, Angular 8, MongoDB, Heroku (backend deployment), GitHub Pages (frontend hosting)
