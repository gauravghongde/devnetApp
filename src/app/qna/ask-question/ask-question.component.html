How to Format
► create code fences with backticks ` or tildes ~ ``` like so ```
► add language identifier to highlight code ```python def function(foo): print(foo) ```
► put returns between paragraphs
► for linebreak add 2 spaces at end
► _italic_ or **bold**
► indent code by 4 spaces
► backtick escapes `like _so_`
► quote by placing > at start of line
► to make links (use https whenever possible) [example](https://example.com) example formatting
